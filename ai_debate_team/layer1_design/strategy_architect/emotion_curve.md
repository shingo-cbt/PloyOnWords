# Emotion Curve — 読者感情設計ガイド

## 感情曲線の基本形

```
感情の振れ幅
↑
強い反発 ──●
            ↘
             ↘  揺らぎ
              ●────●
                        ↘
                         ● 部分同意
                              ↘
                               ● 沈黙（降伏）
→ ターン進行
```

## 各フェーズの定義と設計指針

### Phase 1: エントリー（ターン1）
**目標感情**: 「は？何これ」「ありえない」
- 読者が強くツッコみたくなる出だしにする
- EverymanVoiceが大きな声で反論できる余白を作る
- ここで納得させようとしない（まだ早い）

### Phase 2: 揺さぶり（ターン2-3）
**目標感情**: 「うーん」「でも...」
- 読者の確信に小さなヒビを入れる
- 「正しいとは思わないけど、反論しにくい」状態へ
- 感情的共鳴系の詭弁が有効

### Phase 3: 裏切りポイント（ターン3-4）
**目標感情**: 「え、待って」「そこから来るか」
- 読者が予期しなかった角度からの一撃
- EverymanVoiceが言葉に詰まる瞬間を作る
- ここが最大の山場

### Phase 4: 畳み掛け（ターン4-6）
**目標感情**: 「（口を閉じる）」「なんか...まぁ」
- 論理的に追い詰めるのではなく、「反論のコストが高くなった」状態を作る
- 読者が「もういいか」と思える余白を残す

### Phase 5: 余韻（最終ターン後）
**目標感情**: 「なんか納得したくなかったけど」「モヤっとする」
- 完全な「すっきり」にしない
- このモヤモヤが記憶に残る = コンテンツの成功

## 失敗パターン

### 早期降伏型
- EverymanVoiceが2ターン以内に折れる
- 読者が「弱すぎる」と感じてしまう
- **修正**: EverymanVoiceの抵抗レベルをL3以上に保つ

### 平行線型
- 最後まで両者が歩み寄らない
- 読者が「結局どっちが正しいの」となる
- **修正**: ターン4以降に必ず揺らぎを入れる

### 完全勝利型
- SophistryEngineが論破しすぎる
- 読者が「なんか不快」になる
- **修正**: CognitiveTunerが60/40バランスを維持
