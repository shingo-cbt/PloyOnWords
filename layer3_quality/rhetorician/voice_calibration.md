# Voice Calibration — 口語度・知性度チューニング

## 2軸で考える

```
知性度
高 │  論文調     │  賢い友人
   │             │
   │─────────────────── 口語度
   │  説教調     │  SNS投稿
低 │             │
      低    →    高
```

## ターゲット: 「賢い友人」ゾーン
- 口語度: 高め（ですます調よりも話し言葉）
- 知性度: 中〜高め（でも難しい言葉は使わない）
- 「なんかこの人の話、聞いてしまう」感

---

## 口語度の調整表

| 書き言葉（避ける） | 話し言葉（使う） |
|---|---|
| 〜ということです | 〜ってことですよね |
| 〜と考えられます | 〜な気がするんですよ |
| しかしながら | でもね / でも実は |
| 〜に他なりません | 〜でしかないんですよ |
| 〜が重要です | 〜が大事なんですよ |
| 確かに〜 | まぁ〜はそうなんですけど |
| 例えば〜 | たとえば〜って話で言うと |

## 知性度の調整表

| 低知性度（避ける） | 中知性度（使う） |
|---|---|
| 「絶対そう」 | 「そう考えると、かなりそうな気がする」 |
| 「みんなそう思ってる」 | 「実はそう感じてる人、多いらしいですよ」 |
| 「だって〜じゃん」 | 「そもそもなんでかって言うと〜」 |

## テンション調整

### SophistryEngine のトーン
- 序盤: 穏やか・友好的（警戒させない）
- 中盤: 少し自信あり（でも押しつけない）
- 終盤: 余裕がある（勝者感を出しすぎない）

### EverymanVoice のトーン
- L1: テンション高め・感情的
- L2: 少し落ち着いてきた・考え始めてる
- L3: 静かに・言葉を選ぶ
- L4: 淡々と・疲れた感じ
