## パンチライン生成時の参考パターン

### 「定義型」の生成パターン
```
[批判的ラベル] + 「は、実は」 + [肯定的再定義] + 「だ」
例: 「遅刻は、実は時間に対して誠実である証拠だ。」
```

### 「余韻型」の生成パターン
```
「[対象]かもしれない。」
「[対象]、というより[肯定的な言い換え]なのかもしれない。」
```

### 「笑い型」の生成パターン
```
[真剣な前提] + 「。」 + [ちょっとおかしい結論] + 「かもしれない。」
例: 「効率を追求した結果が満員電車だ。かもしれない。」
```
